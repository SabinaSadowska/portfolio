Tripcity App jest efektem pracy 4-osobowego zespołu. Moim zadaniem było stworzenie wyszukiwarki oraz podłączenie sekcji ulubionych do Firebase, tak, żeby ulubione atrakcje uzytkownika były wyświetlane tylko po zalogowaniu. Technologie: React, JavaScript, HTML, CSS, Firebase, Semantic UI React, Styled Components, React Router.

Tripcity landing page jest projektem wykonanym w 4-osobowym zespole. Moim zadaniem było stworzenie sekcji "O nas" wraz z animacjami CSS oraz sekcji "Zapisz się", w której to uzytkownik po zostawieniu adresu email bedzie mógł zagrać w grę niespodziankę. Moje zadania w grze: unimożliwienie bohaterowi przechodzenia przez granice planszy, wyświetlanie zdobytych punktów oraz wiadomości wyświetlane po zakończeniu rundy. Technologie: CSS, Javascritp, HTML
